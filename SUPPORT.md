# Support

Use [GitHub Issues](https://github.com/xptea/VGPU-Mon/issues) for reproducible bugs and compatibility reports. Please include `vgpu --version`, the exact command, Windows version, GPU model, driver version, and terminal name.

Run `vgpu --demo` to separate terminal rendering problems from GPU-provider problems. `vgpu --json` is useful for collecting a single telemetry snapshot, but review it before posting because process names and GPU identifiers may be sensitive.

Feature ideas belong in a feature request. Usage questions can use GitHub Discussions if discussions are enabled later.

Report vulnerabilities privately using [SECURITY.md](SECURITY.md), never in a public support issue.
